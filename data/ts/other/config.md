# Compiling
```sh
# compile using tsconfig.json
tsc

# compile using my_conf.json
tsc -p my_conf.json
```

## include & exclude
`nano tsconfig.json`
```json
{
    "compilerOptions": {
        "module": "system",
        "noImplicitAny": true,
        "removeComments": true,
        "preserveConstEnums": true,
        "outFile": "../../built/local/tsc.js",
        "sourceMap": true
    },
    "include": ["src/**/*"],
    "exclude": ["node_modules", "**/*.spec.ts"]
}
```
