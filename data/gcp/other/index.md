## deploy
Cloud Deploy is a managed CI/CD service meant to automate the delivery and promotion of application artifacts (like those used in Cloud Run) through various stages (e.g., test, staging, production).
```bash
gcloud deploy --help
```


## dns
Manage your Cloud DNS managed-zones and record-sets.
```bash
gcloud dns --help
```


## domains
Manage domains for your Google Cloud projects.
```bash
gcloud domains --help
```

