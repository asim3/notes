..sub_title..
Create JSON Package
..code..
npm init -y

..text..
Add Express Web Framework
..code..
npm i express
npm install express --save


..sub_title..
run any script from package.json
..code..
"scripts": {
  "start": "node index.js",
  "my_name": "echo 'running my own scripts'",
  "div": "nodemon index"
},

..text..
Run the script
..code..
npm start
npm run my_name
npm run div
node index.js
node index

..text..
install nodemon to restart the server after every save
..code..
npm i -D nodemon

..text..
run
..code..
nodemon index.js
npm run div

..text..
The starting point is the "main" value in 'package.json'
..code..
"main": "index.js",
