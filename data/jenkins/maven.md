# Maven

> Maven is a build automation tool used primarily for `Java projects`.


## install
```txt
Global Tool Configuration:
  Maven:
    Add Maven:
      name: asim-maven-test-1
      install automatically: yes
      version: 3.9.2
        Save


install jenkins plugin: Maven Integration
```
