..sub_title..
Configuration Management


..text..
installing ansible
..code..
sudo add-apt-repository ppa:ansible/ansible
sudo apt-get update
sudo apt install ansible

ansible --version


..text..
configurations
..code..
sudo adduser admin_ansible
sudo chown -R admin_ansible:admin_ansible /etc/ansible
sudo apt upgrade python python3 aptitude


..text..
inventory
..code..
nano /etc/ansible/hosts


..text..
test connection
..code..
ansible -m ping all


..sub_title..
ad-hoc commands "shell commands"
..code..
ansible -m shell -a 'hostname' all
ansible -m shell -a 'whoami' all

ansible -m shell -b -a 'whoami' all 
# Missing sudo password

ansible -m shell -b -K -a 'whoami' all
# root
