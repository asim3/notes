Containerization


..text..
List all images
..code..
sudo docker images


..text..
Image
..code..
sudo docker image rm __id__
sudo docker image pull __id__
sudo docker image run __id__
sudo docker image history __id__



..text..
Containers
..code..
# List all running containers
sudo docker ps 
# List all containers
sudo docker ps -a
# Remove one or more containers
sudo docker rm __id__

sudo docker pause __id__
sudo docker unpause __id__

sudo docker start __id__
sudo docker stop __id__
sudo docker kill __id__



..text..

..code..

..text..
login to a container
..code..
