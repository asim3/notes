## list all installed
```py
sudo snap list
```


## search
```py
sudo snap find firefox

sudo snap info firefox
```


## install
```py
sudo snap install firefox
```


## update
```py
sudo snap refresh firefox

# update all
sudo snap refresh
```


## delete
```py
sudo snap remove firefox
```


## App Aliases
```py
# show all
sudo snap aliases

snap alias --help
snap prefer --help
snap unalias --help
```
