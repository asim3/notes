## Data Types
```php
<?php
  
  var_dump('Hello');
  // string(5) "Hello"

  var_dump("Hello");
  // string(5) "Hello"
  
  var_dump(123);
  // int(123)
  
  var_dump(12.3);
  // float(12.3) 

  var_dump(true);
  // bool(true)

  var_dump(NULL);
  // NULL
  
  var_dump([1, 2, 3]);
  // array(3) { [0]=> int(1) [1]=> int(2) [2]=> int(3) }
  
  var_dump($my_value);
```
