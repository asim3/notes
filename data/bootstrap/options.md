## المساحات
```plaintext
m 		>> margin
p 		>> padding
h 		>> height
w 		>> width
```

## الجهه
```plaintext
t 			>> margin-top or padding-top
b 			>> margin-bottom or padding-bottom
l 			>> margin-left or padding-left
r 			>> margin-right or padding-right
x 			>> both *-left and *-right
y 			>> both *-top and *-bottom
blank 	>> a margin or padding on all 4 sides of the element
```

## الطول
```plaintext
0 		-  margin or padding 	= 0
1 		-  margin or padding 	= $spacer * .25
2 		-  margin or padding 	= $spacer * .5
3 		-  margin or padding 	= $spacer
4 		-  margin or padding 	= $spacer * 1.5
5 		-  margin or padding 	= $spacer * 3
auto 	-  margin 				  	= auto
```
